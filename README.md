# G2FX

Nord G2 Modular editor application

Logic ported from [nord-g2-editor](https://github.com/BVerhue/nord_g2_editor) and [g2ools](https://github.com/msg/g2ools)

Install and Run
===

Currently only builds for Mac Apple Silicon (M1 etc) and Windows 11 X86-64 (Windows 10 might work?)
are generated/supported. 

Grab the appropriate zip from releases, extract and run the app or exe.

### Mac De-quarantine
After unzipping, the app will need to be unquarantined to run. The easiest way to do this is expand the
zip in your Downloads folder, run Terminal, and issue `xattr -cr ~/Downloads/G2FX.app`.
